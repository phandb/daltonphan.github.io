Bug Tracking System

This Bug Tracking System (BTS) was built to provide developer tips and sources of how to solve an issue while developing an application. I used Java Spring Boot and Hibernate as back-end, Thymeleaft engine as front-end. MySQL was used to construct database. The BTS is protected by Spring Security with custom user entity. There are two major roles, Developer and Admin, with different authorizations.
